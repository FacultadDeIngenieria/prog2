Instructions

You have to solve a Second grade equation of the form ax2 + bx + c.

For this create a class named Quadratic (in a new file) that get created from 3 Float numbers, each representing the coefficients a, b, and c.

This class have all this behaviour:

  • roots returns a list of numbers that represents the (x1 and x2).
    • If the equation has only one real solution then it should return one root.
    • If the equation has no real solutions then return an empty List.
  • toString should return a String representing the Quadratic as a String the form Y = A * X2 + B * X + C
  • derivation should return a String representing the derivative of the Quadratic function.

None of these methods receive any parameters, they should use the classes variables for their calculations and output.

Examples

To test your code, you can create a new class called QuadraticTest in a new file and test if the Quadratic methods work correctly, inside a main method.

Quadratic q = new Quadratic(1,-5,6)
q.roots() should return [2,3]
q.toString() should return 1 * x^2 - 5 * x + 6
q.derivation() should return 2 * x - 5
