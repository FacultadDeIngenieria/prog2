Ejercicios Introductorios Java

Se puede utilizar una única clase para crear todas estas funciones. (utilizar una función main donde se ejecuten y verifique que funcionen correctamente).

También, pueden utilizar funciones matemáticas disponibles en la clase Math y utilidades de la clase String

  1. Función Suma: Escribe una función que sume dos números.
  2. Suma de Tres: Modifica la función anterior para que sume tres números.
  3. Promedio de Array: Crea una función que calcule el promedio de un array de números.
  4. Texto en Mayúsculas: Diseña una función que convierta una cadena de texto a mayúsculas.
  5. El Mayor de Tres: Implementa una función que encuentre el mayor de tres números.
  6. Suma Acumulativa: Escribe una función que sume números de 1 a N, donde N es el parámetro.
  7. Factorial: Crea una función que calcule el factorial de un número. (secuencialmente con loops).
  8. Número Par o Impar: Diseña una función que determine si un número es par o impar.
  9. Cuenta Caracteres: Escribe una función que cuente el número de veces que aparece un carácter en una cadena.
  10. Tabla de Multiplicar: Implementa una función que muestre la tabla de multiplicar de un número. (hasta el 10)
  11. Conversión de Temperatura: Crea una función que convierta grados Celsius a Fahrenheit.
  12. Fibonacci: Implementa una función que retorne el enésimo número de Fibonacci. (secuencialmente con loops).
  13. Inversión de Cadena: Escribe una función que invierta una cadena de texto.
  14. Distancia Entre Dos Puntos: Escribe una función que calcule la distancia entre dos puntos en el plano cartesiano.
  15. Área de un Círculo: Crea una función que calcule el área de un círculo dado su radio.

Respuestas

Estas respuestas son una de múltiples posibles respuestas para cada ejercicio, no es necesario que lleguen exactamente a la misma solución. Si prueban la función y realiza lo que indica el enunciado es correcta. Se adjuntan las respuestas para tener una referencia de una posible forma de resolverlo en Java.

// 1. Función Suma
public static int suma(int a, int b) {
    return a + b;
}

// 2. Suma de Tres
public static int sumaDeTres(int a, int b, int c) {
    return a + b + c;
}

// 3. Promedio de Array
public static double promedio(int[] numeros) {
    int suma = 0;
    for (int num : numeros) {
        suma += num;
    }
    return suma / (double) numeros.length;
}

// 4. Texto en Mayúsculas
public static String aMayusculas(String texto) {
    return texto.toUpperCase();
}

// 5. El Mayor de Tres
public static int elMayorDeTres(int a, int b, int c) {
    if (a > b && a > c) return a;
    else if (b > a && b > c) return b;
    else return c;
}

// 6. Suma Acumulativa
public static int sumaAcumulativa(int N) {
    int suma = 0;
    for (int i = 1; i <= N; i++) {
        suma += i;
    }
    return suma;
}

// 7. Factorial
public static int factorial(int n) {
    int resultado = 1;
    for (int i = 1; i <= n; i++) {
        resultado *= i;
    }
    return resultado;
}

// 8. Número Par o Impar
public static boolean esPar(int n) {
    return n % 2 == 0;
}

// 9. Cuenta Caracteres
public static int cuentaCaracteres(String texto, char caracter) {
    int contador = 0;
    for (int i = 0; i < texto.length(); i++) {
        if (texto.charAt(i) == caracter) {
            contador++;
        }
    }
    return contador;
}

// 10. Tabla de Multiplicar
public static void tablaDeMultiplicar(int n) {
    for (int i = 1; i <= 10; i++) {
        System.out.println(n + " * " + i + " = " + (n * i));
    }
}

// 11. Conversión de Temperatura
public static double celsiusAFahrenheit(double celsius) {
    return (celsius * 9/5) + 32;
}

// 12. Fibonacci Secuencial
public static int fibonacciSecuencial(int n) {
    if (n <= 1) return n;
    int a = 0, b = 1, c;
    for (int i = 2; i <= n; i++) {
        c = a + b;
        a = b;
        b = c;
    }
    return b;
}

// 13. Inversión de Cadena
public static String invertirCadena(String cadena) {
    return new StringBuilder(cadena).reverse().toString();
}

// 16. Distancia Entre Dos Puntos
public static double distanciaEntrePuntos(double x1, double y1, double x2, double y2) {
    return Math.sqrt(Math.pow(x2 - x1, 2) + Math.pow(y2 - y1, 2));
}

// 17. Área de un Círculo
public static double areaCirculo(double radio) {
    return Math.PI * Math.pow(radio, 2);
}
